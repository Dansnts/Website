Backup Automation

Solution de sauvegarde automatisée avec suivi des erreurs

Vue d'ensemble

Solution de backup automatisée utilisant rclone pour synchroniser les données vers différents backends de stockage cloud. Le système inclut un suivi complet des logs et une gestion intelligente des erreurs.

Fonctionnalités

  • Synchronisation incrémentale avec rclone
  • Support multi-destinations (S3, Google Drive, etc.)
  • Logging détaillé de chaque opération
  • Suivi et alertes sur les erreurs
  • Politique de rétention configurable
  • Rapports de backup par email

Configuration rclone

Fichier de configuration rclone definissant le remote Google Drive. Le token OAuth est stocke de maniere securisee (secret Kubernetes ou fichier protege).

rclone.conf INI 
[gdrive]
type = drive
scope = drive
token = {"access_token":"...","token_type":"Bearer","refresh_token":"...","expiry":"..."}
root_folder_id = 1xXxXxXxXxXxXxXxXxXxXxXxXxXxXx

[nas]
type = smb
host = 192.168.1.240
user = backup-user
pass = OBFUSCATED_PASSWORD
domain = WORKGROUP
backup.sh Bash 
#!/bin/bash
set -euo pipefail

LOCKFILE=/tmp/backup-nas.lock
LOG=/var/log/backup/rclone-$(date +%Y%m%d).log
SRC=/mnt/nas
REMOTE_GD="gdrive:backup/nas"
RETENTION_DAYS=30

# Eviter les executions concurrentes
if [ -f "$LOCKFILE" ]; then
  echo "$(date): Job deja en cours, sortie." >> "$LOG"
  exit 0
fi
trap 'rm -f "$LOCKFILE"' EXIT
touch "$LOCKFILE"

echo "$(date): Debut backup NAS -> cloud" >> "$LOG"

# Sync vers Google Drive
if rclone sync "$SRC" "$REMOTE_GD" \
    --log-file="$LOG" \
    --log-level INFO \
    --stats 1m \
    --transfers 4 \
    --checkers 8 \
    --bwlimit 50M; then
  echo "$(date): Google Drive OK" >> "$LOG"
else
  echo "$(date): ERREUR Google Drive (exit $?)" >> "$LOG"
  # Notification via webhook Slack
  curl -s -X POST "$SLACK_WEBHOOK" \
    -H 'Content-type: application/json' \
    --data "{\"text\":\"Backup NAS->GDrive FAILED sur $(hostname)\"}"
fi

# Politique de retention : supprimer fichiers de plus de 30 jours
find /var/log/backup/ -name "rclone-*.log" -mtime +"$RETENTION_DAYS" -delete

echo "$(date): Backup termine" >> "$LOG"

Gestion des erreurs

  • Parsing automatique des logs rclone
  • Classification des erreurs (réseau, permission, espace disque)
  • Retry automatique avec backoff exponentiel
  • Alertes email/Slack en cas d'échec
