Physical to Virtual : Pipeline d'automatisation réseau Aruba AOS-CX vers EVE-NG

Problématique

Tester des modifications sur un réseau de production Aruba est risqué et coûteux. Les erreurs de configuration peuvent entraîner des interruptions de service. Comment valider des changements sans impacter l'infrastructure existante ?

Pipeline

1

Collecte

Ansible collecte les configs running et les tables LLDP de chaque switch Aruba AOS-CX.

→
2

Nettoyage

ArubaConfigCleaner supprime les commandes non supportées en virtuel (VSF, UBT, DHCP) et remappe les ports uplink.

→
3

Topologie

LLDPToUNL parse les tables LLDP, déduplique les liens, et génère le fichier UNL avec les configs en base64.

→
4

Import EVE-NG

Le ZIP généré s'importe directement dans EVE-NG, topologie et configs ready-to-boot.

ArubaConfigCleaner.py : Pipeline de nettoyage

Les switches physiques Aruba utilisent des fonctionnalités non disponibles sur les images virtuelles : VSF (stacking), UBT, PoE, et des ports uplink numérotés 1/1/47-49. Le cleaner adapte automatiquement chaque config.

ArubaConfigCleaner.py Python 
# Remapping ports physiques uplink → virtuels (limite 9 ports max)
PORT_MAPPING = {
    "1/1/47": "1/1/8",
    "1/1/48": "1/1/9",
    "1/1/49": "1/1/7"
}

# Patterns de lignes supprimées (non supportées en virtuel)
SKIP_PATTERNS = [
    r"user\s+admin\s+group\s+administrators\s+password\s+ciphertext",
    r"active-gateway l3-src-mac",
    r"client track ip",
    r"ip source-interface ubt",
    r"replication-mode",
    r"^ip dhcp$",
    r"^no ipv6 dhcp$"
]

def modifier_contenu(texte, vxlan_config=None):
    """Pipeline complet : extraction → VSF → lignes → ports → VXLANs."""
    version = extraire_version(texte)
    config  = extraire_config(texte)       # strip Ansible wrapper
    config  = supprimer_vsf(config)         # remove VSF member blocks
    config  = filtrer_lignes(config)        # remove SKIP_PATTERNS lines
    config  = remapper_interfaces(config)   # 1/1/47→1/1/8, etc.
    config  = filtrer_vxlans(config, vxlan_config)  # limit to 32 VXLANs

    return f"!Version AOS-CX {version}\nauto-confirm\nconfigure terminal\n\n{config}\n"

LLDPToUNL.py : Parsing topologique

Depuis les fichiers LLDP collectés, le script reconstruit la topologie complète, déduplique les liens bidirectionnels, et génère un fichier XML/UNL avec les configs encodées en base64 directement embarquées.

LLDPToUNL.py Python 
def remap_port(self, sw, port):
    """Remappe les ports physiques hors plage vers virtuels (max 1/1/9)."""
    m = re.match(r'(\d+/\d+/)(\d+)', port)
    if m:
        if int(m.group(2)) <= 9:
            return port  # déjà dans la plage virtuelle
        if port in PORT_MAP:
            self.remapped[(sw, port)] = PORT_MAP[port]
            return PORT_MAP[port]
    return port

def parse_file(self, filepath):
    """Parse un fichier LLDP et extrait les connexions switch↔switch."""
    with open(filepath, 'r', encoding='utf-8') as f:
        content = f.read()

    # Nom du switch local depuis le nom de fichier : lldp-SWB-CHE-LAB-1.conf
    m = re.search(r'lldp-(.+)\.conf', os.path.basename(filepath))
    local_sw = self.add_switch(m.group(1))

    for block in re.split(r'-{20,}', content):
        port_m  = re.search(r'^Port\s+:\s+(\S+)', block, re.MULTILINE)
        name_m  = re.search(r'Neighbor System-Name\s+:\s+(.+)', block)
        nport_m = re.search(r'Neighbor Port-ID\s+:\s+(\S+)', block)

        if not (port_m and name_m and nport_m):
            continue

        local_port   = port_m.group(1)
        neighbor     = name_m.group(1).strip()
        neighbor_port = nport_m.group(1).strip()

        if local_port == 'mgmt' or neighbor in ['6300', 'GS1350']:
            continue  # ignorer ports mgmt et équipements non-Aruba

        lp = self.remap_port(local_sw, local_port)
        np = self.remap_port(neighbor, neighbor_port)
        self.connections.append((local_sw, lp, neighbor, np))

def normalize_connections(self):
    """Déduplique les liens bidirectionnels (A→B == B→A)."""
    unique = set()
    for s1, p1, s2, p2 in self.connections:
        key = (s1, p1, s2, p2) if (s1, p1) < (s2, p2) else (s2, p2, s1, p1)
        unique.add(key)
    return sorted(unique)

P2V.py : Orchestration du workflow

Script principal qui enchaîne les 4 étapes, gère les dossiers temporaires, et produit le ZIP prêt à l'import EVE-NG.

P2V.py Python 
def run(self):
    """Workflow complet : validation → nettoyage → UNL → ZIP."""
    try:
        if not self.validate_input():
            sys.exit(1)

        converted = self.convert_configs()   # → ArubaConfigCleaner
        self.generate_unl(converted)          # → LLDPToUNL
        zip_path  = self.create_zip()          # → .zip pour EVE-NG

        print(f"UNL : {os.path.abspath(self.output_unl)}")
        print(f"ZIP : {os.path.abspath(zip_path)}")
        print("Import EVE-NG : File → Import → Browse → .zip")

    except Exception as e:
        print(f"ERREUR: {e}")
        sys.exit(1)

    finally:
        self.cleanup()  # supprime les fichiers temporaires


# Usage
# python3 P2V.py ./data/ output.unl
