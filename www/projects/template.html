Déploiement

Script utilisé pour builder et déployer l'application.

deploy.sh Bash 
#!/bin/bash
# Script de déploiement automatisé

set -euo pipefail

APP_DIR="/opt/mon-app"
IMAGE="mon-app:latest"

echo "→ Pull des dernières modifications..."
git -C $APP_DIR pull origin main

echo "→ Build de l'image Docker..."
docker build -t $IMAGE $APP_DIR

echo "→ Redémarrage du container..."
docker compose -f $APP_DIR/compose.yaml up -d --force-recreate

echo "✓ Déploiement terminé."

Exemple de code applicatif

monitor.py Python 
import requests
from datetime import datetime

def check_service(url: str) -> bool:
    """Vérifie si un service HTTP répond correctement."""
    try:
        resp = requests.get(url, timeout=5)
        return resp.status_code == 200
    except requests.exceptions.RequestException:
        return False

if __name__ == "__main__":
    services = ["http://localhost:8080", "http://localhost:9090"]
    for svc in services:
        status = "✓ UP" if check_service(svc) else "✗ DOWN"
        print(f"[{datetime.now():%H:%M:%S}] {svc} — {status}")
