ROOT

Zero-knowledge personal workspace : Notes, Kanban, Calendrier

Contexte

ROOT est un workspace personnel entièrement local et chiffré, conçu pour ne jamais exposer de données à un serveur. L'objectif : avoir un espace de travail complet (notes, tâches, agenda) sans dépendre d'un service cloud, sans compte, sans télémétrie.

Modules

1

Notes

Éditeur de blocs riche (TipTap 3) : titres, listes, code, tableaux, images. Arborescence de pages avec drag-and-drop.

→
2

Kanban

Board de tâches persistant avec colonnes de statut, priorités, dates d'échéance et tags. Drag-and-drop via dnd-kit.

→
3

Calendrier

FullCalendar 6 avec sync CalDAV bidirectionnelle (iCloud, Nextcloud, Fastmail). Le proxy nginx local garantit que les données ne quittent jamais la machine.

Architecture de sécurité

  • Dérivation de clé PBKDF2-SHA256, 600 000 itérations, sel aléatoire 256 bits
  • Chiffrement AES-GCM 256 bits avec IV unique 96 bits par ciphertext
  • Le mot de passe maître n'est jamais stocké, génère une CryptoKey non-extractible en RAM
  • Tout ce qui est écrit dans IndexedDB (titres, contenu, réglages, credentials) est du ciphertext
  • HTTPS obligatoire (Web Crypto API), servi par nginx en local sur port 8443
