PiCamera

Appareil photo DIY avec Raspberry Pi - Projet de pré-apprentissage ETML (2015)

Contexte

Projet réalisé lors de mon pré-apprentissage à l'ETML en 2015. L'objectif était de concevoir un appareil photo autonome à partir d'un Raspberry Pi 3B, capable de prendre des photos via un écran tactile et de les stocker puis les synchroniser vers Dropbox.

Architecture matérielle

Raspberry Pi 3B Carte principale, sans WiFi intégré à l'époque
Camera Module Module caméra officiel Raspberry Pi
Écran tactile Adafruit PCB Adafruit avec écran TFT tactile pour l'interface
Carte SD Stockage local des photos avant upload

Extrait du code Python

Logique principale de capture et d'upload. La bibliotheque picamera controle le module camera, pygame gere l'interface sur l'ecran TFT Adafruit.

picamera_app.py Python 
import picamera
import pygame
import dropbox
import os
import time
from datetime import datetime

# Configuration
DROPBOX_TOKEN = os.environ.get('DROPBOX_TOKEN')
PHOTO_DIR = '/home/pi/photos'
PREVIEW_DURATION = 3  # secondes

def init_display():
    """Initialise l'ecran TFT Adafruit 2.8" via pygame."""
    os.environ['SDL_VIDEODRIVER'] = 'fbcon'
    os.environ['SDL_FBDEV'] = '/dev/fb1'  # framebuffer TFT
    pygame.init()
    screen = pygame.display.set_mode((320, 240))
    pygame.mouse.set_visible(False)
    return screen

def capture_photo(camera, filename):
    """Capture une photo avec preview sur l'ecran."""
    camera.start_preview()
    time.sleep(PREVIEW_DURATION)  # autofocus + exposition
    camera.capture(filename)
    camera.stop_preview()
    return filename

def upload_dropbox(local_path, remote_path):
    """Upload vers Dropbox via API v2."""
    dbx = dropbox.Dropbox(DROPBOX_TOKEN)
    with open(local_path, 'rb') as f:
        dbx.files_upload(
            f.read(),
            remote_path,
            mode=dropbox.files.WriteMode.add
        )

def main():
    os.makedirs(PHOTO_DIR, exist_ok=True)
    screen = init_display()

    with picamera.PiCamera() as camera:
        camera.resolution = (2592, 1944)  # 5 MP full resolution
        camera.rotation = 180

        running = True
        while running:
            for event in pygame.event.get():
                if event.type == pygame.MOUSEBUTTONDOWN:
                    # Bouton tactile appuye : prendre une photo
                    ts = datetime.now().strftime('%Y%m%d_%H%M%S')
                    filename = f"{PHOTO_DIR}/photo_{ts}.jpg"

                    screen.fill((0, 0, 0))
                    capture_photo(camera, filename)

                    # Upload en arriere-plan
                    remote = f"/PiCamera/photos/photo_{ts}.jpg"
                    upload_dropbox(filename, remote)

                    # Afficher confirmation sur ecran
                    font = pygame.font.Font(None, 24)
                    txt = font.render("Sauvegarde OK", True, (0, 255, 0))
                    screen.blit(txt, (80, 110))
                    pygame.display.flip()
                    time.sleep(1)

if __name__ == '__main__':
    main()

Logiciel

  • Programme en Python pour le contrôle de la caméra et l'interface tactile
  • Stockage des photos sur la carte SD
  • Upload automatique vers Dropbox via connexion Ethernet (pas de WiFi sur la Pi 3B de 2015)
  • Interface utilisateur sur l'écran tactile Adafruit

Fabrication

  • Conception d'une coque sur mesure en 3D
  • Impression 3D du boîtier
  • Assemblage de tous les composants (Pi, caméra, écran, PCB)
