Stack de monitoring complète pour infrastructure

Vue d'ensemble

Solution de monitoring centralisée basée sur Prometheus et Grafana. Collecte de métriques, visualisation et alerting pour garder un oeil sur l'infrastructure.

Flux de collecte Prometheus

Prometheus scrute chaque cible selon un intervalle configuré. node-exporter expose les métriques système, cAdvisor les métriques containers. Alertmanager achemine les alertes vers les canaux de notification.

Configuration Prometheus

Fichier de configuration principal definissant les intervalles de scrape, les cibles statiques et les regles d'alerte chargees depuis des fichiers externes.

prometheus.yml YAML 
global:
  scrape_interval:     15s
  evaluation_interval: 15s
  external_labels:
    monitor: 'homelab'

alerting:
  alertmanagers:
    - static_configs:
        - targets: ['alertmanager:9093']

rule_files:
  - /etc/prometheus/rules/*.yml

scrape_configs:
  - job_name: 'prometheus'
    static_configs:
      - targets: ['localhost:9090']

  - job_name: 'node'
    static_configs:
      - targets:
          - '192.168.1.16:9100'   # k8s node Ubuntu
          - '192.168.1.240:9100'  # NAS FreeNAS (si node-exporter installe)
    relabel_configs:
      - source_labels: [__address__]
        target_label: instance

  - job_name: 'cadvisor'
    static_configs:
      - targets: ['cadvisor:8080']

  - job_name: 'blackbox-http'
    metrics_path: /probe
    params:
      module: [http_2xx]
    static_configs:
      - targets:
          - https://root.fariadossantos.com
    relabel_configs:
      - source_labels: [__address__]
        target_label: __param_target
      - source_labels: [__param_target]
        target_label: instance
      - target_label: __address__
        replacement: blackbox-exporter:9115

Regles d'alerte

Exemple de regles Prometheus pour surveiller la disponibilite des noeuds et l'utilisation des ressources.

rules/node-alerts.yml YAML 
groups:
  - name: node
    rules:
      - alert: NodeDown
        expr: up{job="node"} == 0
        for: 2m
        labels:
          severity: critical
        annotations:
          summary: "Noeud {{ $labels.instance }} injoignable"

      - alert: HighCPUUsage
        expr: 100 - (avg by(instance) (rate(node_cpu_seconds_total{mode="idle"}[5m])) * 100) > 85
        for: 5m
        labels:
          severity: warning
        annotations:
          summary: "CPU eleve sur {{ $labels.instance }}: {{ $value | printf \"%.1f\" }}%"

      - alert: DiskSpaceLow
        expr: (node_filesystem_avail_bytes{mountpoint="/"} / node_filesystem_size_bytes{mountpoint="/"}) * 100 < 15
        for: 10m
        labels:
          severity: warning
        annotations:
          summary: "Espace disque faible sur {{ $labels.instance }}: {{ $value | printf \"%.1f\" }}% libre"

      - alert: HighMemoryUsage
        expr: (1 - (node_memory_MemAvailable_bytes / node_memory_MemTotal_bytes)) * 100 > 90
        for: 5m
        labels:
          severity: warning
        annotations:
          summary: "RAM elevee sur {{ $labels.instance }}: {{ $value | printf \"%.1f\" }}%"
