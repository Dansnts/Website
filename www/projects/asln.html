ASLN Website

Site web dynamique pour l'Association Sport et Loisirs Nestlé

Contexte

Projet réalisé durant mon stage chez Nestlé. L'ASLN (Association Sport et Loisirs Nestlé) avait besoin d'un site web simple à maintenir pour présenter les différents clubs sportifs et de loisirs proposés aux employés.

Défi

Le site devait être maintenable par des personnes sans connaissances techniques. Ajouter un nouveau club ne devait pas nécessiter de modifier le code existant.

Solution

J'ai conçu un système dynamique où l'ajout d'un club se fait simplement en déposant deux fichiers :

1

nom_club.html

Page de présentation du club avec le contenu souhaité.

→
2

logo_club.png

Logo du club pour l'affichage sur la page d'accueil.

→
3

Auto-détection

Le site détecte automatiquement les nouveaux clubs et les affiche.

Bénéfices

  • Aucune compétence technique requise pour ajouter un club
  • Pas de modification du code source nécessaire
  • Convention de nommage simple et intuitive
  • Maintenance minimale
