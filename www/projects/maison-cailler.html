Maison Cailler - Ateliers Chocolat à Distance

Mise en place d'ateliers chocolat interactifs à distance pendant le COVID-19

Contexte

Durant la pandémie de COVID-19, Maison Cailler a dû réinventer ses ateliers chocolat. Le concept : les clients recevaient une box à domicile contenant tous les ingrédients nécessaires, puis suivaient en direct les maîtres chocolatiers via Zoom pour réaliser leurs créations. J'ai été chargé de mettre en place toute l'infrastructure technique pour rendre ces ateliers interactifs possibles.

Défis

  • Permettre aux maîtres chocolatiers de guider les clients en direct depuis l'atelier
  • Gérer des sessions avec de nombreux participants simultanés
  • Assurer une qualité vidéo suffisante pour montrer les gestes techniques
  • Prévoir une solution de backup en cas de problème technique

Solution déployée

Zoom Plateforme principale pour les ateliers en direct avec les clients
Microsoft Teams Solution de backup en cas de problème avec Zoom
OneDrive / SharePoint Stockage cloud pour le partage de documents et recettes
Box d'ingrédients Envoi aux clients avec tous les ingrédients pour suivre l'atelier
